# 📝 To-Do List

Uma aplicação completa de gestão de tarefas, permitindo que cada utilizador tenha a sua própria lista protegida por autenticação. 

## Tecnologias Utilizadas

### **Backend**
* **Java 17** & **Spring Boot 3**
* **Spring Security**
* **JWT (JSON Web Tokens)**
* **Spring Data JPA**
* **MySQL**
* **Maven**

### **Frontend**
* **React** (utilizando **Vite**)
* **JavaScript (ES6+)**
* **Fetch API**
* **CSS In-JS**

---

## Como rodar o projeto

### **1. Pré-requisitos**
* **JDK 17** ou superior instalado.
* **MySQL Server** em execução.
* **Node.js** e **NPM** instalados.
